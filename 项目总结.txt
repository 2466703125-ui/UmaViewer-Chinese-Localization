UmaViewer 中文本地化改动总结
================================

完成日期：2026-04-29

一、核心问题：TMP 字体缺少中文字形（方框根因）
----------------------------------------------

问题：
- 导航栏标题显示为 □□
- 原因：TextMeshPro 的 SDF 字体图集默认只包含 ASCII + 拉丁字符

解决方案：
1. 创建 ChineseChars.txt 中文字符集文件
2. 用 Font Asset Creator 生成中文图集
3. 设置 Fallback 字体或替换字体引用

二、代码修复
-----------

1. AutoLocalizeUI.cs 修复：
   - 添加 textToKeyCache 缓存，避免重复生成 key
   - 修改 ProcessTextComponent，检查是否已被翻译
   - 删除 IsAllChinese 方法

2. 新增工具：
   - ExtractChineseChars.cs - 提取中文字符集
   - CleanDuplicateKeys.cs - 清理重复翻译 Key

四、验证流程
-----------

1. 运行 Tools → 提取中文字符集 → 生成 ChineseChars.txt
2. 用 Font Asset Creator 生成含中文的 TMP 字体图集
3. 将新字体设为 Fallback 或替换场景中的字体引用
4. 运行 Tools → 清理重复翻译 Key → 清理 zh_cn.json
5. 运行 Tools → 自动本地化 UI → 重新扫描添加 LocalizedLabel

7. 检查所有面板标题、按钮、标签是否都有中文
